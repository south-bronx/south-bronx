## 🐛 Common Issues and Fixes

### ❌ Script executes but GUI doesn't appear
**Fix:** Press `RightShift` or `Insert` to manually toggle the GUI. If still invisible, disable Discord overlay, GeForce Experience overlay, or any active ReShade filters — then re-inject and execute.

### ❌ Auto Farm stops collecting after a few minutes
**Fix:** Toggle Auto Farm off and on to reset the pathing and detection loop. If the issue persists, a game update likely changed the farm object positions — check for a script update.

### ❌ Silent Aim not registering hits
**Fix:** This usually means a game update changed the hit detection method. Update to the latest script version immediately — Silent Aim is the first module patched after any South Bronx combat update.

### ❌ INF Money not reflecting in game balance
**Fix:** The INF Money module works on the client's local money variable. Some server-side validations may reject extreme jumps. Use the "Increment Mode" in the GUI settings to add money in smaller, realistic amounts that pass server validation.

### ❌ Script not working on Android
**Fix:** Ensure you're using **Delta v2.3+** or **Arceus X v2.0+** and that Roblox was launched through the executor's built-in launcher. The **south bronx script auto farm mobile** module requires the game to be fully loaded before injection — wait for the spawn screen before executing.

### ❌ ESP not showing players
**Fix:** ESP renders after the game server fully loads player character models (~5 seconds after joining). Wait for the round to start before toggling ESP. If still blank, toggle it off and on to reinitialize the render loop.

### ❌ "attempt to index nil value" error on execution
**Fix:** A recent game update broke a variable reference in the script. Check the issues tab of this repository or the **south bronx script discord** announcements channel for a hotfix, or wait for the next scheduled patch.
